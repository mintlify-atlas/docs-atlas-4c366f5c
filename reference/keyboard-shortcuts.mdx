Keyboard shortcuts let you keep your hands on the keyboard and move faster through every stage of pipeline building — from placing and wiring nodes to navigating large canvases and managing your edit history. The shortcuts below cover every supported action in GUIness.

## Node operations

| Key | Action |
|-----|--------|
| `Delete` | Delete selected node(s) |
| `Shift+D` | Duplicate selected node(s) |
| `Ctrl+Z` | Undo |
| `Ctrl+Shift+Z` | Redo |
| `Ctrl+C` | Copy selected node(s) |
| `Ctrl+V` | Paste node(s) |

## Canvas navigation

| Key | Action |
|-----|--------|
| `Spacebar + Drag` | Pan the canvas |
| `Scroll Wheel` | Zoom in/out |

## Selection

| Key | Action |
|-----|--------|
| `Ctrl+A` | Select all nodes |
| `Escape` | Deselect all |
| `Alt+Click` | Select node + all downstream |
| `Shift+Click` | Add/remove from selection |
| `Click+Drag` (empty canvas) | Lasso select |

<Tip>
Use `Alt+Click` on any node to instantly select it and every downstream node connected to it. This is the fastest way to grab an entire processing chain before moving, duplicating, or deleting it.
</Tip>
